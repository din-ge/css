რა არის CSS

HTML არის დაკაბადონების ენა, CSS კი მისი ვიზუალური გაფორმება. მათი ერთობლივი გამოყენებით შესაძლებელია ნებისმიერი გამოხატულების საიტის შექმნა

მაგალითისთვის:

შესაძლებელია ტეგ body-ში ყველა შრიფტის ზომა იყოს 14, ამასთან h1 სათაურს ჰქონდეს ზომა 24 და ფერი წითელი, ხოლო h4 სათაურებს ზომა 12 და ფერი მწვანე. ვებ-გვერტის ამ ელემენტების პარამეტრები განსხვავდება, ე. ი. აქვთ განსხვავებული სტილები

ყოველ ობიექტს (აბზაცი, სათაურები, ხაზები, ტექსტი) ვებ-გვედზე შესაძლოა ჰქონდეს თავისი სტილი. ყველა სტილების ნაკრებს ჰქვია სტილების ცხრილი

თუ ობიექტს ეთითება რამოდენიმე სტილი, მაშინ გამოიყენება კასკადირება, რომელიც განსაზღვრავს სტილიში აღწერილი თვისების უპირატესობას

CSS-ის უპირატესობა

  • CSS კოდი ხდება კითხვადი და მცირდება ზომაში
  • CSS გვეხმარება ახალი პარამეტრების შემოტანაში, რომელთა მოქმედება შეუძლებელია მხოლოდ HTML ენით. მაგალითისთვის ბმულისთვის გახავზვის მოშორება.
  • CSS-ით ყოველვის მარტივია საიტის გარეგნობის შეცვლა. მაგალითისთვის დაკაბადონებულია 100 გვერდიანი საიტი, რომლის ყველა სათაური არის ლურჯი. როდესაც საჭირო გახდება სათაურების ფერის შეცვლა, ეს მოხდება CSS-ის ერთი პარამეტრის შეცვლით, რაც სხვა შემთხვევაში, საჭირო იქნებოდა ასივე გვედზე იგივეს გაკეთება.
